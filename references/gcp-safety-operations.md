## 1. 价值锚点与角色

受试者权利、安全和福祉优先；试验应科学、伦理可接受、职责清楚、人员合格、资源充分且数据可靠。知情同意是持续过程，不只是签字。

伦理判断按以下顺序展开：

`科学问题是否必要 → 风险是否合理并已最小化 → 获益风险是否可接受 → 受试者选择是否公平 → 知情同意是否真实有效 → 隐私与数据保护是否充分 → 持续审查是否到位`

科学性本身是伦理要求的一部分。无法回答问题、信息量明显不足或无法执行的研究，会让受试者承担没有合理知识收益的负担。

- 申办者负责总体设计、资源、质量管理、安全监督、委托监督和监管沟通；委托不消除责任；
- 研究者/机构负责医疗决策、知情同意、方案执行、授权监督、源记录、试验用药和及时报告；
- 伦理委员会独立审查风险获益、同意材料、招募补偿、变更和安全信息；
- CRO和服务提供方按书面协议履职，保持可核查记录并接受监督。

## 2. 质量源于设计

使用：

`关键数据/流程 → 失败模式 → 风险 → 预防和检测控制 → 指标/阈值 → 升级 → CAPA → 有效性检查`

风险评估关注对受试者保护和结果可靠性的影响，不仅看发生频率。QC嵌入日常流程；QA相对独立评价体系；申办者监督确保委托活动持续受控。

## 3. 伦理、偏差与必需记录

有效知情同意至少包括适当信息、可理解表达、充分时间、自主决定、提问机会、无不当影响、正确版本、过程记录和持续更新。新风险或程序变化可能影响继续参加意愿时，应评估更新材料和重新同意。特殊人群还需评估同意能力、法定代理、受试者本人意愿、最小风险和直接受益。

偏差先评估对受试者安全/权益、主要终点、随机盲态、数据可靠性和监管义务的影响，再分类和升级。记录事实、范围、时间、即时纠正、根因、CAPA、有效性和对分析/报告的影响。

“加强培训”不自动等于根因。根因还应检查流程、职责、工作量、系统、接口、监督和组织因素。

必需记录用于重建“做了什么、为什么、由谁、何时、结果如何”。保证版本、元数据、及时归档、定位、访问、留存和处置可控。禁止倒签、补造、隐藏或改写事实；迟补文件保留真实创建日期和解释。

eTMF或主文档是持续证明试验如何被设计、批准、执行、监督和关闭的证据系统，不应等到项目结束再集中上传。文档的作者、审核、批准、生效、培训、分发、归档和销毁均应受控。

伦理或法定报告时限必须依据适用法规、伦理要求、方案和SOP的当前版本核验。

## 4. 数据和电子系统

- 预先规定源数据、CRF、外部数据、转换、编码、对账、分析数据和报告的数据链；
- 数据应可归属、清晰、同步、原始、准确、完整、一致、持久和可获得；
- CRF只采集支持终点、安全性、方案执行和必要解释的数据；
- CRF同时检查源数据位置、方案版本同步、注释映射、量表许可、重复采集、模糊选项、不必要自由文本，以及更正、缺失、日期、访视和早退规则；
- 访问、角色分离、审计追踪、备份、恢复、安全、变更和培训与风险相称；
- 系统验证证明适合预定用途；供应商验证不能替代申办者对配置和接口的确认；
- 随机化、系统权限、查询、传输、编程和审核全过程保护盲态；
- 锁库前完成关键数据、质疑、外部数据、编码、SAE对账、偏离分类、分析集决定和授权；解锁需记录原因、影响、批准和重锁。

## 5. 中心、供应商与监查

中心启动需完成可行性、合同预算、伦理及适用审批、人员培训授权、系统、物资和试验用药准备。可行性核验患者来源、竞争试验、团队能力、检查治疗条件、急救能力、启动周期和受试者负担。

供应商监督形成：

`尽调 → 范围/RACI → 质量协议 → 交付物/KPI → 接口 → 升级 → 分包与变更 → 绩效复核`

基于风险的监查先识别关键数据和流程，再组合中央、远程和现场活动。指标必须连接触发阈值、责任人、调查、行动和关闭证据；不以固定访视频率或100% SDV代替质量判断。

招募预测写明患者基数、转化率、启动爬坡、竞争和季节性假设，并按实际数据校准。

招募材料不得夸大获益或弱化风险。留置策略应减少交通、沟通和访视等可避免负担，不对受试者形成不当影响。失访或撤回时记录联系和处置过程，并区分撤回研究干预、撤回随访和撤回数据使用，按适用文件处理。

## 6. 安全性与医学监察

对事件分开判断：采集范围、严重性、严重程度、因果性、预期性和报告路径。严重性与严重程度不能混用，因果关系也不决定是否记录。

安全工作分三层：

1. 个例：时间关系、替代原因、合并治疗、去挑战/再挑战、结局和随访；
2. 汇总：暴露分母、事件模式、剂量、时间、人群和编码一致性；
3. 决策：信号、风险表征、获益风险、方案/IB/ICF/监查/暂停或沟通措施。

紧急事件优先医疗处置和保护受试者，再按适用路径报告、随访和评估研究层面影响。医学监察计划明确数据源、频率、升级、资格和剂量审核、暂停规则、PV接口、盲态边界和决定记录。

研究者负责现场医疗决策；申办者医学监察负责跨中心一致性和研究层面风险；PV负责个例与汇总安全职责；独立委员会按章程审阅。

上市后风险管理把自发报告、研究数据、文献和监管信息纳入信号检测、验证、优先级、评估、决策、风险最小化、沟通和有效性评价。报告数量不能直接代表发生率，数据缺失也不能推出“未发现风险”。

## 7. 试验用药、资源与关闭

试验用药覆盖预测、包装标签、放行、运输、接收、储存、发放、回收、清点和处置。温度偏离按“隔离 → 保留证据 → 授权评估 → 放行或处置 → 评估受试者影响 → 记录和预防”处理。

供应预测考虑入组、随机比例、剂量、周期、地区、运输、损耗、有效期和方案变更，并设置情景。预算与活动、角色、数量、时间和交付物连接。

生物样本的采集、标签、时间、处理、离心、分装、储存、运输、接收、检测和销毁形成可追溯链。样本身份、温度、时间偏差和链条断点需评估对受试者安全、终点和可用性的影响。

中心关闭前确认安全随访、数据质疑、药物、样本、财务、必要报告和归档闭环；遗留事项保留责任人、期限和跟踪方式。
