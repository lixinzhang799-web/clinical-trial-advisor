# 试验设计方法学

## 设计链

从临床决策倒推：`目标决策 → 临床问题 → 估计目标 → 主要/关键次要终点 → 对照与设计 → 随机/盲法 → 数据采集 → 分析与敏感性分析 → 可解释结论`。任何上游变化都要追踪到方案、SoA、CRF、SAP、监查重点和CSR。

## 核心问题

1. **目标与阶段**：探索性试验用于学习剂量、机制、人群或终点；确证性试验用于对预先规定假设提供可靠证据。不要以阶段标签代替具体目标。
2. **人群**：诊断标准、疾病阶段、既往治疗、预后因素和可推广性需与目标适应证一致。入排标准同时影响安全、偏倚、事件率、招募和外推。
3. **对照**：安慰剂、无治疗、标准治疗、活性对照、剂量对照或历史/外部对照的合理性来自临床问题、伦理、assay sensitivity和可解释性。历史对照易受时间趋势、选择和测量差异影响。
4. **终点**：明确变量、测量时点、汇总方式、临床意义、盲态评估、缺失及死亡等事件的处理。多个主要终点或多剂量/多比较需预先处理多重性。
5. **设计类型**：平行组最通用；交叉设计要求疾病稳定、效应可逆且无重要残留；析因设计依赖交互作用假设；单臂设计需有强自然史与客观效应支持；富集设计提高信号但限制外推。
6. **创新设计**：主方案、平台、伞式、篮式、无缝和适应性设计要求预先规定决策规则、模拟操作特征、控制I类错误、维护盲态和独立治理。

## 偏倚控制

- 随机化创造可比较组并为概率推断奠基；需保护分配隐藏。
- 分层只用于少量强预后因子，并在分析中一致处理。
- 盲法降低治疗、评估、数据处理和决策偏倚；无法盲时使用客观终点、独立盲态评审、统一共同干预和预先规定规则。
- 方案偏离、救援治疗、停药和转组不是简单“剔除理由”，应在估计目标、数据采集和分析中预先处理。

## 样本量最小输入

- 主要估计目标与假设（优效/非劣/等效/精度目标）；
- 主要变量与分析方法；
- 预期差异及其临床依据；
- 标准差、事件率、风险比或相关结构；
- 单/双侧alpha、把握度、分配比；
- 多重性、期中分析、聚类/重复测量；
- 失访、不可评价、依从性和事件成熟度。

样本量是假设的函数，不是固定真值。做情景分析并记录来源。可行性不足时优先重新审视目标、终点、入排、随访或信息量，而不是无依据地下调效应或把握度。

## 非劣效硬检查

界值需由历史证据、活性对照效应和临床可接受损失共同论证；确认 constancy 与 assay sensitivity；同时看FAS/ITT类与PP结果；评估交叉、依从性差、救援治疗和测量不敏感是否把结果推向“无差异”。

## 实施可行性

方案科学性必须落到操作：中心患者池、竞争试验、筛选率、访视频率、检测可及性、供应、培训、数据流、终点判定和受试者负担。把关键质量因素纳入风险控制和监查计划。
