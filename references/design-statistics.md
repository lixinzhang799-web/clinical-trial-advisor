# 试验设计与统计

## 设计链

从临床决策倒推：

`目标决策 → 临床问题 → 估计目标 → 人群 → 干预与对照 → 终点 → 随机/盲法 → 数据采集 → 分析与敏感性分析 → 可行性 → 可解释结论`

上游变化必须追踪到方案、SoA、CRF、SAP、监查重点、安全管理和CSR。

## 设计核心问题

1. **目标与阶段**：探索性研究支持学习剂量、机制、人群或终点；确证性研究支持预先规定的临床与监管问题。阶段标签不能替代具体目标。
2. **人群**：诊断、疾病阶段、既往治疗、预后因素和可推广性需与目标适应证一致。入排标准同时影响安全、偏倚、事件率、招募和外推。
3. **对照**：根据临床问题、伦理、试验敏感性和可解释性选择安慰剂、标准治疗、活性、剂量或外部对照。历史/外部对照需控制时间趋势、选择和测量差异。
4. **终点**：明确变量、时间点、汇总方式、临床意义、评估盲态，以及缺失、死亡和救援治疗等事件的处理。复合终点、替代终点、PRO、生物标志物和全局评价还需明确组成、测量工具、事件判定、缺失处理和临床解释。
5. **设计类型**：平行组最通用；交叉设计要求疾病稳定、效应可逆且残留可控；单臂设计需有强自然史和客观效应支持；富集设计提高信号但限制外推。
6. **复杂设计**：主方案、平台、伞式、篮式、无缝和适应性设计需预先规定决策规则、模拟操作特征、错误率控制、盲态和独立治理。

## 方案形成与修订

方案不是单人写作产物。推荐流程：

`开发策略/概念 → 方案纲要 → 跨职能输入 → 设计决策日志 → 可行性验证 → 完整方案 → 下游文件同步 → 培训与放行`

修订时记录原因，并评估对已入组与未来受试者、伦理和监管路径、统计、CRF/EDC/IRT、供应、中心培训、知情同意、版本切换和报告解释的影响。

## 偏倚控制

- 随机化创造可比较组并支持概率推断，同时保护分配隐藏；
- 分层限于少量强预后因子，并在分析中一致处理；
- 盲法覆盖治疗、评估、数据处理和决策角色；无法盲时使用客观终点、独立盲态评审和统一共同干预；
- 方案偏离、停药、救援治疗和转组应进入估计目标、数据采集和分析计划，不是简单剔除理由。

## 估计目标

完整估计目标包含：

1. 治疗条件；
2. 目标人群；
3. 变量或终点；
4. 伴发事件及处理策略；
5. 总体层面的汇总量。

常见伴发事件策略包括治疗策略、假设策略、复合变量、在治策略和主层策略。不同事件可以采用不同策略。策略定义的是临床问题，不等于插补方法。

## 缺失数据、估计和敏感性

伴发事件改变“要估计什么”，缺失数据表示估计该目标所需的数据未被观察。先定义估计目标和应持续收集的数据，再讨论缺失机制、主要估计和敏感性分析。

- 主要估计方法必须与估计目标一致；
- 敏感性分析检验同一估计目标对关键不可验证假设的稳健性；
- 补充分析回答不同问题，不应误称为敏感性分析；
- 停药不等于退出研究，停药后数据可能仍是主要分析所必需。

## 分析集

- 全分析集尽量保留随机化优势，但“ITT”不能代替完整估计目标；
- PP集可支持依从性和非劣效稳健性评价，事后剔除可能产生选择偏倚；
- 安全集通常按实际暴露组织，换组和多次暴露规则需预先规定。

## 样本量最小输入

- 主要估计目标与优效、非劣、等效或精度目标；
- 主要变量、分析方法和预期效应；
- 标准差、事件率、风险比或相关结构；
- 单/双侧alpha、把握度和分配比；
- 多重性、期中分析、聚类或重复测量；
- 失访、不可评价、依从性和事件成熟度。

样本量是假设的函数，不是固定真值。缺关键参数时只给计算框架、参数表和情景分析，不输出伪精确数字。

## 非劣效、多重性与亚组

- 非劣效界值需由历史对照效应和临床可接受损失共同论证；
- 检查constancy、assay sensitivity、依从性、交叉、救援治疗及FAS/ITT类与PP结果；
- 主要终点、剂量、组别、时间点和期中决策共同构成多重性，需预先定义错误率控制；
- 亚组以一致性和交互解释为主，不能用“一个显著、一个不显著”自动证明差异。

## 可行性与交付质量门

设计需落到患者池、中心、访视、检测、供应、培训、数据流、终点判定和受试者负担。交付前检查临床问题、估计目标、终点、数据收集、分析和可行性是否闭合；任何关键假设都应注明来源和不确定性。
